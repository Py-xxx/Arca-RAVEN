# Rovers

USD articulation models for each rover type. Added in Phase 2.

## Planned Models

| ID | Name | Wheels | Based On |
|---|---|---|---|
| `rocker_bogie` | 6-Wheel Rocker-Bogie | 6 | NASA Curiosity / Perseverance |
| `skid_steer` | 4-Wheel Skid-Steer | 4 | Generic utility rover |
| `tracked` | Tracked Rover | — | Future phase |

## Format
- USD (Universal Scene Description) — Isaac Sim native
- URDF available for ROS2 bridge compatibility
