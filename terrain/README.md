# Terrain

USD scene files for each terrain environment. Added in Phase 2.

## Presets

| ID | Environment | Gravity | Notes |
|---|---|---|---|
| `mars` | Gale Crater, Mars | −3.72 m/s² | Basalt regolith, SCM soil model |
| `moon` | Mare Tranquillitatis | −1.62 m/s² | Fine-grained regolith, deep dust |
| `asteroid` | 433 Eros | −0.003 m/s² | Micro-gravity, anchoring dynamics |
| `earth` | Mojave Desert, CA | −9.81 m/s² | Mixed soil, rock debris |
| `custom` | User-defined | configurable | GeoTIFF heightmap import |
